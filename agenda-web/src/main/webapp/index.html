Exemplo de sistema web com Servlets e JSP

Exemplo de listagem e inclusão de contatos no sistema

  • Usa JavaDB como SGBD. Instruções para a criação do banco e o script para criação da tabela está no diretório META-INF.
  • Para login, usar os usuários "fulano" ou "ciclano", e a senha "abcd1234" (tudo sem aspas). O "fulano" tem permissões de administrador.
  • Neste exemplo, os usuários estão "mockeados", ou seja, não é feita consulta no banco de dados para autenticar o usuário.
  • Notar que, ao tentar acessar a lista de contatos sem login (mesmo colando a URL correta no browser), o sistema força um redirecionamento para a tela de login.
  • O contrário também é válido, ou seja, se o usuário logado tentar acessar a tela de login, o sistema o redireciona para a tela de listagem de contatos
  • Os arquivos css estão em /resources/css.
